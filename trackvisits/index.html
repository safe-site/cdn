Script Demo

Welcome to the demo page! This page demonstrates a script that tracks user visits and sends notifications to a Telegram bot.

How it works

When you visit this page, the script prompts you for your email. If you provide a valid email, it will be stored in the local database along with visit statistics. If you cancel the prompt or provide an invalid email, a dummy username will be generated and used.
